4 - plug0

6 - plug2
7 - control power
8 - server power 2
9 - server power 3
10 - TX
11 - RX
12 - server power 1
13 - server power 4
14 - server reset 1
15 - server reset 2
16 - server reset 3
17 - server reset 4
18 - server reset 5
19 - server power 5
